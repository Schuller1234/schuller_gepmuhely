# Fémtech Kft. Weboldal - HTML/CSS/JS Export

Ez a mappa tartalmazza a Fémtech Kft. weboldalának tiszta HTML/CSS/JavaScript verzióját.

## Fájlok

- **index.html** - A fő HTML fájl, tartalmazza az összes tartalmat
- **styles.css** - Az összes stílus egy helyen
- **script.js** - JavaScript funkciók (menü, smooth scroll, űrlap kezelés)
- **images/** - Mappa a generált képekkel

## Használat

1. Nyisd meg az `index.html` fájlt böngészőben
2. Vagy töltsd fel egy webszerverre/hostingra
3. A képek az `images/` mappában találhatók

## Funkciók

- ✅ Teljesen reszponzív (mobil és desktop)
- ✅ Smooth scroll navigáció
- ✅ Mobil hamburger menü
- ✅ Kapcsolati űrlap (szimulált küldés)
- ✅ Modern animációk
- ✅ SEO optimalizált

## Böngésző támogatás

Minden modern böngészőben működik (Chrome, Firefox, Safari, Edge).

## Testreszabás

- **Színek:** A `styles.css` fájl `:root` szekcióban találod a színváltozókat
- **Szövegek:** Az `index.html` fájlban szerkesztheted a szövegeket
- **Képek:** Cseréld ki a képeket az `images/` mappában (ugyanolyan névvel)
